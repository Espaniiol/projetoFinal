Exercício 17: Galeria de Imagens com Filtro

Descrição: Construa uma galeria de imagens com categorias (ex: natureza, tecnologia, pessoas). Crie botões para filtrar e mostrar apenas imagens da categoria selecionada.

Como foi resolvido: Cada imagem possui um atributo `data-category`. Botões de filtro escutam cliques e, com base na categoria selecionada, adicionam ou removem uma classe `hidden` das imagens, utilizando transições CSS para suavizar o efeito.
